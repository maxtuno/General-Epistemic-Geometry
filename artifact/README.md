# Supplementary Artifacts

This directory stores supplementary materials associated with the canonical documents in this repository.

Typical contents may include:
- extended derivations and technical appendices,
- protocol notes and audit checklists,
- additional tables or intermediate computational exports,
- replication bundles that complement `code/`, `data/`, and `figures/`.

All files placed here should remain traceable, versioned, and aligned with the repository evidence contract.
